  • Codacy

    • 最近使い始めたコード解析ツール．

    • Pull requestも大丈夫か解析してくれる模様（今のところDependabotからのPull requestにしか適用したことがありませんが...）

    • 多彩なlinterを駆使してVisual Stadio Code上で捉えきれなかった部分を指摘してくれることがあり，とても助かっています．

  • Microsoft Copilot

    • 参考文献をしっかり見ることが出来る，無料でウェブ検索も行ってくれる，検索専門よりはある程度自由に動かせられるということで主戦力の1つ．

    • 独特の会話のノリが見てて面白かったのですが，学術的なのをぶつけすぎて真面目になってしまったのか最近はあまりなくなってしまいました．

    • しばらくはかつてPhindでやっていた軽いプロンプトだったのですが，今は正確性や会話の流れの確認を極めようと試行錯誤し初手の指示プロンプトが大変なことに...

  • Claude

    • 最近こっちの方がいいのかもとか思っていたり．

    • 出典もきれいに指示に従ってくれるし，論文の読解力やエラー対応もかなりイケてる感じ．

    • えらい厳しい無料枠の制限さえなければ...というところです．
